## Install

### Manual

From melpa, `M-x package-install RET all-the-icons-ibuffer RET`.

``` emacs-lisp
(all-the-icons-ibuffer-mode 1)
```

### Use-package

``` emacs-lisp
(use-package all-the-icons-ibuffer
  :ensure t
  :init (all-the-icons-ibuffer-mode 1))
```

Enjoy! :smile:
